# План разработки проекта IMock

Этот план составлен с учетом методологии **NoDemo** (код сразу готов к продакшену) и принятых технических решений:
*   **Backend:** Python, FastAPI, SQLite (Repository Pattern), JWT, Yandex Cloud AI.
*   **Frontend:** TypeScript, React, Material UI (MUI).
*   **Comms:** REST API + Short Polling.
*   **Infra:** Docker Compose.
